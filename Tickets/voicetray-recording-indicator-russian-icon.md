# VoiceTray: индикатор записи, русский вывод и нормальная иконка приложения

## Контекст

По результатам проверки MVP обнаружены три UX/ASR-замечания:

1. После нажатия `Start` пользователь говорит голосом, но на экране нет заметного признака, что запись идет. Текст появляется только после `Stop`, поэтому нужен анимированный индикатор записи в стиле ChatGPT/voice recording.
2. После распознавания текст получается не на русском, а в транслитерации, например: `Dobre, utrowsiem. Okej.` Нужно настроить распознавание на русский язык.
3. У приложения сейчас дефолтная системная иконка. Нужно добавить приличную иконку приложения и tray icon, желательно с мотивом микрофона.

## Требования

- Добавить визуальный индикатор состояния записи в главном окне.
- Индикатор должен быть заметен во время записи и скрываться/успокаиваться после `Stop`.
- Не делать streaming recognition на этом этапе: текст по-прежнему может появляться после `Stop`.
- Настроить whisper.cpp на русский язык по умолчанию.
- Обновить `settings.example.json` и дефолтные настройки.
- Добавить app icon `.ico`.
- Использовать эту иконку для окна приложения, exe и tray icon.
- Обновить README после реализации.

## Acceptance Criteria

1. При нажатии `Start` пользователь видит анимированный индикатор записи.
2. При нажатии `Stop` индикатор записи отключается, статус переходит в распознавание.
3. По умолчанию whisper.cpp запускается с русским языком.
4. Распознанный русский текст выводится кириллицей, если модель и аудио корректны.
5. Окно приложения и tray icon используют кастомную иконку VoiceTray.
6. `dotnet build` проходит без ошибок.
7. `README.md` обновлен.
